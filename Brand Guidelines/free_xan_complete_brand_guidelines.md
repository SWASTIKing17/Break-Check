# 01 — Brand Overview

## Brand Name
# freeXan

## Parent Brand
# BloomX

## Product Category
Creative Workflow Automation Software

## Product Description
freeXan is an intelligent workflow automation system for video editors and creative teams.

It automates:
- Premiere Pro project setup
- Folder structure generation
- Media organization
- Drag-and-drop imports
- Creative workflow acceleration
- Background synchronization systems

freeXan is designed as invisible infrastructure for modern creators.

The product should feel:
- Fast
- Intelligent
- Minimal
- Premium
- Creative-tech focused
- System-driven
- Modern
- Powerful without complexity

- balanced geometric proportions

# 11 — UI Design Language

# Core Principles

## 1. Minimal Friction
Every interaction should feel instant.

## 2. Focus on Flow
The UI should disappear while working.

## 3. Soft Futurism
Modern without becoming overly sci-fi.

## 4. Motion as Feedback
Movement should indicate:
- progress
- sync
- automation
- responsiveness

## 5. Layered Depth
Use:
- blur
- transparency
- soft shadows
- glow

# 13 — Motion Language
