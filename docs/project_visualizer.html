This interactive diagram maps the connection channels, message protocols, and components within freeXan.

Key Inter-process flows
  • WebSocket protocol (Port 4554): Syncs Premiere's project path to main.js, and passes imported files back to CEP.
  • IPC bridge: Protects app safety by isolating renderer files while allowing controlled file drops, sizing, and configuration.
  • PowerShell monitor: Polls Premiere's window title as a native fallback to read the active project folder structure.
